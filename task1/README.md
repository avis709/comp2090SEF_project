LEADERBOARD SYSTEM
This is a program where we apply concepts including Abstraction, Encapsulation, Modularity, Abstract Data Type and such in the classes when setting the data structure.

function:
record name, score, house with the register() module
check the record with the check() module
compare the user's score with other people with preset data and comparison()

Error prevention:
the steps loops when invalid input was detected(ex. inputting non extisting house or negative score)
suggest possible fixes(write writting record that exceed human limits in long jump, suggest user to change the score

how to use:
download classes.py and main.py
put them in the same folder
run main.py
